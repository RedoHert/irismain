Qual API nós usamos?

Nós utilizamos a API da Groq, e o nosso modelo de linguagem utiliza o modelo Mixtral, que é um modelo otimizado para responder perguntas e realizar alguns tipos de tarefas.

A Iris AI é segura?

Sim, pois nós não utilizamos nenhum tipo de Database em nossos serviços, e isso inpede de que outros usuários possam modificar os seus dados. Nós não utilizamos nenhum tipo de serviços que armazenam dados, assim tornando a nossa IA mais segura.

A Iris AI é rápida?

Sim, ela possui um modelo otimizado para realizar vários tipos de ações, e possui diversos parâmetros fornecidos pela Groq, e Mixtral.

É fácil de usar?

A Iris AI possui uma interface amigável e intuítiva. Ela possui uma área para modificar o modelo, de acordo com sua escolha. Ela também permite que você escolha quantos Tokens de repostas você quiser.

A Iris AI é open-source?

A Iris Ai já foi open-source, mas no momento, o seu repositório foi definido para privado, por questões de segurança. Porém, nós pretendemos torná-la open-source novamente em uma hora ou outra.
